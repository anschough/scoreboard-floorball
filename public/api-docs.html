Stream Deck shortcuts

Klicka på en shortcut nedan för att kopiera hela URL:en till urklipp. Denna URL ska sedan läggas till i Stream Deck via funktionen System → Website. Glöm inte att markera "GET request in background" när du klistrat in din URL.

Bas-URL http://localhost:3000
